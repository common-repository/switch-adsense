Show/hide Google adsense on individual page, post, category, all pages or posts.

== Description ==
With Adsense Switch, you can enable/disable Google Adsense for:
all pages
all posts
all post with selected categories
individual page
individual post

== Installation ==
Same as other plugins, upload & install Adsense Switch and activate.
Document is avaialble [here](https://www.ino-kawa.com/?page_id=1302)
